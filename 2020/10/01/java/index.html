递归

实际开发中，递归可以解决具有既定规律的特定问题。

何时递归

  • 有固定的规律，函数中自己调用自己。
  • 当需要解决的问题可以分成若干的小问题且大小问题的解决方法相同。

如何正确使用递归

有效的出口条件，避免无穷递归。

阶乘、1-100相加，1-100输出

public class TestMultiply {
    public static void main(String[] args) {
        int num = 5;
        int result = multiply(num);
        System.out.println(result);

        System.out.println(add(100));

        prints(100);
    }

    public static int multiply5(int n) {
        return n * multiply4(n - 1);
    }

    public static int multiply4(int n) {
        return n * multiply3(n - 1);
    }

    public static int multiply3(int n) {
        return n * multiply2(n - 1);
    }

    public static int multiply2(int n) {
        return n * multiply1(n - 1);
    }

    public static int multiply1(int n) {
        return 1;
    }

    public static int multiply(int n) {
        if (n == 1) {
            return 1;
        }
        return n * multiply(n - 1);
    }

    public static int add(int n) {
        if (n == 1) {
            return 1;
        }
        return n + add(n - 1);
    }

    public static void prints(int n) {
        System.out.println(n);
        if (n == 1) {
            return;
        }
        prints(n - 1);
    }
}

冒泡排序

相邻个两个值相互比较大小，互换位置。

public class Bubbleordor {
    public static void main(String[] args) {
        int[] nums = { 4, 3, 5, 2, 1 };
        // 第一轮
        for (int j = 0; j < nums.length - 1; j++) {
            if (nums[j] > nums[j + 1]) {
                int temp = nums[j];
                nums[j] = nums[j + 1];
                nums[j + 1] = temp;
            }
        }
        printArray(nums);
        // 第二轮
        for (int j = 0; j < nums.length - 1 - 1; j++) {
            if (nums[j] > nums[j + 1]) {
                int temp = nums[j];
                nums[j] = nums[j + 1];
                nums[j + 1] = temp;
            }
        }
        printArray(nums);
        // 第三轮
        for (int j = 0; j < nums.length - 1 - 1 - 1; j++) {
            if (nums[j] > nums[j + 1]) {
                int temp = nums[j];
                nums[j] = nums[j + 1];
                nums[j + 1] = temp;
            }
        }
        printArray(nums);
        // 第四轮
        for (int j = 0; j < nums.length - 1 - 1 - 1 - 1; j++) {
            if (nums[j] > nums[j + 1]) {
                int temp = nums[j];
                nums[j] = nums[j + 1];
                nums[j + 1] = temp;
            }
        }
        printArray(nums);

        // 最终综合
        for (int i = 0; i < nums.length - 1; i++) {
            for (int j = 0; j < nums.length - 1 - i; j++) {
                if (nums[j] > nums[j + 1]) {
                    int temp = nums[j];
                    nums[j] = nums[j + 1];
                    nums[j + 1] = temp;
                }
            }
        }
        printArray(nums);
    }

    public static void printArray(int[] nums) {
        for (int i = 0; i < nums.length; i++) {
            System.out.print(nums[i] + " ");
        }
        System.out.println();
    }
}

选择排序

固定值与其他值比较大小，互换位置。

public class SelectSort {
    public static void main(String[] args) {
        int[] nums = { 4, 3, 5, 2, 1 };
        // 第一轮
        int i = 0;
        for (int j = 1; j < nums.length; j++) {
            if (nums[i] > nums[j]) {
                int temp = nums[i];
                nums[i] = nums[j];
                nums[j] = temp;
            }
        }
        printArray(nums);
        // 第二轮
        i++;
        for (int j = i + 1; j < nums.length; j++) {
            if (nums[i] > nums[j]) {
                int temp = nums[i];
                nums[i] = nums[j];
                nums[j] = temp;
            }
        }
        printArray(nums);
        //最终综合
        for (i = 0; i < nums.length - 1; i++) {
            for (int j = i + 1; j < nums.length; j++) {
                if (nums[i] > nums[j]) {
                    int temp = nums[i];
                    nums[i] = nums[j];
                    nums[j] = temp;
                }
            }
        }
        printArray(nums);
    }

    public static void printArray(int[] nums) {
        for (int i = 0; i < nums.length; i++) {
            System.out.print(nums[i] + " ");
        }
        System.out.println();
    }
}

冒泡和选择的区别 :

（1）冒泡排序是比较相邻位置的两个数，而选择排序是按顺序比较，找最大值或者最小值；

（2）冒泡排序每一轮比较后，位置不对都需要换位置，选择排序每一轮比较都只需要换一次位置；

（3）冒泡排序是通过数去找位置，选择排序是给定位置去找数；

冒泡排序优缺点：

  优点:比较简单，空间复杂度较低，是稳定的；

  缺点:时间复杂度太高，效率慢；

选择排序优缺点：

  优点：一轮比较只需要换一次位置；

  缺点：效率慢，不稳定（举个例子5，8，5，2，9 我们知道第一遍选择第一个元素5会和2交换，那么原序列中2个5的相对位置前后顺序就破坏了）。
